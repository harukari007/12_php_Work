# OpenAI Playground

## 操作説明
1.身長、体重、年齢を登録すると、BRMを自動生成
2.生成されたカロリーと、嫌いな食べ物を除外して、ChatGPTがレシピを作成してくれます。


## 仕様
1.ログイン
2.基本情報を登録
3.食事情報を登録
4.レシピ自動生成


## スクリプト
1. `script.js` GPT-3.5を使った対話
1. `weatherBot.js` 関数呼び出し機能を使った対話
1. `tokenCounter.js` トークン数の出力
1. `gpt4oImageProcessing.js` 食事の画像をからカロリーを計算する対話
